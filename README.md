# SQL-Data-Ware-House-Project

Welcome all!
My first project in sql. This demonstates a comprehensive data warehousing and analytics solution, from building a ware house to generating actionable insite.

## Project Requirement
### Building a modern data warehouse with SQL server including ELT processes, data modelling and analytics
#### Objective
Develop a modern data warehouse using SQL server to consolidate sales, enabling analytical reporting and infored decision-making.
#### Specifications
- **Data Sources** : Import data from two source system {ERP and ARM} provided as CSV file.
- **Data Quality** : Cleans and resolve data quality issue prior to analysis.
- **Integration** : Combine both source into single, user-friendly data model designed for analytical queries.
- **Scope** : Focus on the latest dataset only; historization of data is not recognized.
- **Documentation** : Provide clear documentation of the data model to suppose both business stake holders and analytics team.
### BI : Analytic and Reporting {Data Analytics}
#### Objective
Develop SQL-based analytics to deliver detailed insight intro:
- **Customer Behaviour**
- **Product Performance**
- **Sales Trends**
